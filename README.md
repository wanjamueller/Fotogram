A responsive photo gallery created as part of my Full-Stack Engineering training at Developer Akademie.

Fotogram presents a collection of photographs in a clean, modern interface and allows users to browse the gallery in an intuitive and visually engaging way.

About the Project

This project was developed during my current training to become a Full-Stack Engineer. Its purpose was to strengthen my practical understanding of front-end development by building an interactive website from scratch.

While creating Fotogram, I focused on:

* Writing semantic and well-structured HTML
* Creating a responsive layout with CSS
* Using JavaScript to add interactive behavior
* Working with arrays, functions, loops, events, and DOM manipulation
* Organizing project files and reusable code
* Improving the user experience across different screen sizes

Features

* Responsive photo-gallery layout
* Interactive image viewing
* Navigation through the image collection
* Clean and user-friendly design
* Support for desktop, tablet, and mobile screens
* Dynamically rendered gallery content

Technologies Used

* HTML5
* CSS3
* JavaScript
* Git
* GitHub

Getting Started

No installation or build process is required.

1. Clone the repository:

git clone https://github.com/YOUR-USERNAME/YOUR-REPOSITORY.git

2. Open the project folder:

cd YOUR-REPOSITORY

3. Open index.html in your browser.

For development, the project can also be opened with a local development server such as the Live Server extension for Visual Studio Code.
